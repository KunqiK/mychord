❓ Chord HUD 使用指南

这是一个为视频叠加设计的和弦 HUD（1920×1080，黑底，叠加模式用 Screen）。下面按使用顺序讲清每个功能——新手从「快速上手」开始即可。

① 快速上手（4 步）
  1. 填基本信息：左侧设置 BPM、调号、拍号、曲名等，点 『应用基本信息』。
  2. 编辑时间轴：右侧点 『＋ 添加和弦』，填写每个和弦的出现「时间」和「和弦名」，并在「高亮音级」里点亮要强调的音。
  3. 预览：点顶部 ▶ 预览 全屏播放（再按任意键停止；播放时点进度条可跳转）。
  4. 导出：点 ⬇ MP4（推荐，兼容 DaVinci Resolve）。导入剪辑软件后把图层叠加模式设为 Screen，黑底即会透明。
② 时间轴每行的字段
  • 时间(s)该和弦出现的时间点（秒）。可切换为「小节·拍」模式输入。
  • 和弦名显示的和弦文字，如 Cmaj7、Am、BbMaj9。
  • 承接前同时把前 N 个和弦点亮的音级一起显示（琥珀色），当前和弦自身的音级为紫色。用 − / + 调整，便于展示和声的延续。填了分组的和弦会自动计算：等于它前面连续同组和弦的数量。
  • 分组填入相同分组名的相邻和弦会同时显示（如斜杠 / 叠置和弦）；当前演奏的最大，其余缩小淡显。留空＝独奏。
  • 高亮音级12 个音级方格，点击切换该和弦要点亮的音。
  • 解说显示在和弦下方的一行注释（和声功能、分析等）。
  • ★重要和弦：开启后用红色光晕脉冲强调。
  • ☑ / ✕左侧勾选框配合「复制所选 / 粘贴到末尾」批量复制；✕ 删除该行。
  • 也可能是MIDI 导入时给出的备选和弦，点一下即可替换当前和弦名。
③ 时间模式与拍号
  • 秒 ↔ 小节·拍：点左侧 『小节模式』 可用「第几小节·第几拍」输入；内部仍按 BPM 自动换算为秒。
  • 拍号按钮（2/4 – 7/4 或自定义）决定开头每小节拍数；改 BPM / 拍号会自动重算小节模式下的时间。
  • 数拍子 / 拍号变化（设计页）：HUD 顶部会显示节拍指示器（当前拍高亮、每小节第一拍琥珀色）。若歌曲中途变拍，在此点『＋ 添加拍号变化』，填「从第几小节起 · 每小节几拍」；小节·拍的换算与数拍子都会随之改变。可取消勾选隐藏指示器。
④ 导入素材
  • 📋 粘贴导入：粘贴 Am G F C 或 | Am | G | F | C | 之类文字，按设定的「每和弦占几拍」自动生成时间轴。
  • 🎹 MIDI 导入：读取 .mid 文件，自动识别和弦（含备选和弦与转位/斜杠低音）。
  • 背景音频：作为配乐，在预览和导出的视频里一起播放。
  • 背景图片 / 视频：作底图绘制在 HUD 之后，可勾选暗化遮罩。
⑤ 导出格式怎么选
  • ⬇ MP4：H.264，推荐，兼容 DaVinci Resolve 等剪辑软件；已加载的背景音频会一并写入（AAC）；导出速度快于实时。
  • ⬇ WebM：VP9，体积小但部分软件（DaVinci Windows 版）不支持；按实际时长实时录制。

导出的视频为黑底，叠加到画面上时请把图层混合 / 叠加模式设为 Screen，黑色即变透明。

⑦ 小贴士

  • 音名区可自定义 12 个半音的显示名（如改成唱名 Do Re Mi）。
  • 和弦垂直位置滑块调整和弦在画面中的高度。
⑧ v12 新功能
  • 🖥 实时预览：可拖动的悬浮小窗，边编辑边看效果（与导出画面完全一致）。拖动下方滑杆跳到任意时间，▶ 从该位置播放（含音频）。点 ⧉ 弹出 变成独立窗口，可拖到浏览器外或另一块屏幕。
  • 音级双行显示：7 个调内音级放大显示在上排，5 个调外（变化）音级缩小放在下排空隙处，类似钢琴黑键布局。
  • 旋转唱片：左侧加载专辑封面后，音级下方出现以封面为主的旋转唱片；转速与 BPM 同步（每 1/2/4/8 小节转一圈），预览与导出都会包含。

A chord HUD made to overlay on video (1920×1080, black background, blend mode Screen). The features below are in usage order — newcomers can start with "Quick start".

① Quick start (4 steps)
  1. Fill in the basics: on the Input page set BPM, key, time signature, title, etc., then click Apply info.
  2. Edit the timeline: click ＋ Add chord, set each chord's time and name, and light up the notes to emphasize under "Highlight degrees".
  3. Preview: click ▶ Preview to play fullscreen (any key stops; click the progress bar to jump).
  4. Export: click ⬇ MP4 (recommended, works in DaVinci Resolve). In your editor set the layer's blend mode to Screen so the black turns transparent.
② Fields on each timeline row
  • Time (s)When the chord appears (seconds). Can switch to "Bar·Beat" mode.
  • ChordThe chord text shown, e.g. Cmaj7, Am, BbMaj9.
  • CarryAlso shows the degrees of the previous N chords (amber); the current chord's own degrees are purple. Use − / + to adjust. Grouped chords auto-calculate it: the count of consecutive preceding chords in the same group.
  • GroupAdjacent chords with the same group name show together (e.g. slash / stacked chords); the playing one is largest, others shrink & dim. Empty = solo.
  • Degrees12 degree cells — click to toggle which notes light up.
  • NoteA one-line annotation under the chord (harmonic function, analysis, etc.).
  • ★Emphasize: a red glow pulse highlights the chord.
  • ☑ / ✕The checkbox works with "Copy selected / Paste to end" for bulk copy; ✕ deletes the row.
  • AlsoMIDI import offers alternative chords; click one to replace the current name.
③ Time mode & time signature
  • Seconds ↔ Bar·Beat: click Bars to enter "bar · beat"; internally still converted to seconds via BPM.
  • Time-signature buttons (2/4 – 7/4 or custom) set the opening beats per bar; changing BPM / signature re-computes times in bar mode.
  • Beat counter / meter changes (Design page): a beat indicator shows at the top of the HUD (current beat highlighted, beat 1 in amber). If the song changes metre partway, click ＋ Add meter change and fill in "from bar N · beats per bar"; both the bar·beat conversion and the beat counter follow the change. Untick to hide the indicator.
④ Importing material
  • 📋 Paste: paste text like Am G F C or | Am | G | F | C |; a timeline is generated from the set "beats per chord".
  • 🎹 MIDI: reads a .mid file and auto-detects chords (with alternatives and inversions / slash bass).
  • Background audio: plays as a soundtrack in preview and export.
  • Background image / video: drawn behind the HUD; optional dark overlay.
⑤ Choosing an export format
  • ⬇ MP4: H.264, recommended, works in DaVinci Resolve etc.; loaded background audio is written in too (AAC); faster than real time.
  • ⬇ WebM: VP9, smaller but unsupported by some software (DaVinci on Windows); recorded in real time.

The exported video is on a black background; when overlaying, set the layer's blend / composite mode to Screen so black becomes transparent.

⑥ Shortcuts

Ctrl+Z Undo · Ctrl+Y Redo · Space Play / pause the live preview · ← → Seek the live preview ±5s (Shift ±25s · Ctrl ±2s) · Any key stops fullscreen preview · Click progress bar to jump · Esc closes dialogs

⑦ Tips
  • 💾 Save / 📂 Load: save the whole project (info + timeline) as a .json to continue later.
  • The Note names area lets you rename the 12 semitones (e.g. to solfège Do Re Mi).
  • The Chord vertical position slider adjusts the chord's height on screen.
⑧ v12 features
  • 🖥 Live preview: a docked preview that matches the export exactly. Drag the slider to any time, ▶ plays from there (with audio). Click ⧉ Pop out for a separate window you can drag outside the browser / to another screen.
  • Two-row degrees: the 7 in-key degrees are enlarged on top, the 5 out-of-key (chromatic) ones shrink into the gaps below, like a piano's black keys.
  • Rotating disc: load an album cover and a cover-dominant vinyl appears below the degrees; speed syncs to BPM (one turn per 1/2/4/8 bars), included in preview and export.
